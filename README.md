# CTANET

A little game like <a href="http://cursors.io">cursors.io</a>.

## Getting Start

### Install

```cmd
git clone https://github.com/swwind/ctanet.git
cd ctanet
npm install
```

### Run

```cmd
npm start
```

or

```cmd
node app.js
```

or

```cmd
run-server.cmd
```

Now you can play with your friends in the LAN.
